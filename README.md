# Kvalifikācijas eksāmens 
Šeit tiks aprakstīts mans kvalifikācijas eksāmena projekts

## Projekta apraksts

SIA **"Ražots Godiņos"** ir Latvijas kokapstrādes un galdniecības uzņēmums, kas specializējas dažādu koka būvju izgatavošanā un uzstādīšanā.

Uzņēmums piedāvā:

- Lapenes
- Auto nojumes
- Terases
- Saimniecības ēkas
- Koka nojumes
- Dārza mājiņas
- Individuāli projektētas koka konstrukcijas
- Koka izstrādājumu montāžu un uzstādīšanu


## Projekta mērķis

Šī kvalifikācijas darba ietvaros tiek izstrādāta tīmekļa vietne uzņēmumam **Ražots Godiņos**, lai uzlabotu uzņēmuma digitālo klātbūtni internetā, nodrošinātu ērtu informācijas pieejamību klientiem un demonstrētu uzņēmuma paveiktos darbus.

## Galvenās funkcijas

- Informācija par uzņēmumu
- Pakalpojumu katalogs
- Paveikto projektu galerija
- Kontaktu informācija
- Saziņas forma klientiem
- Responsīvs dizains dažādām ierīcēm
- Cenu kalkulators
